Scaffool 🚀

Scaffool is my personal backend API scaffolding tool for Node.js projects. It automates setup so I don’t have to do repetitive tasks every time.

📥 Installation:

	npm install -g scaffool

⚡ Features:

	📦 Automatic installation of required packages
	
	📝 Optional TypeScript setup
	
	📂 Creates standard backend folder structure automatically


Basic Usage:

	scaffool build <framework>
	
-- Creates folder structure

-- Initializes project

-- Installs dependencies

-- Optionally sets up TypeScript

Example:

	scaffool build express

📝 Notes

-- Designed for quick backend API project setup

-- Handles conditional TypeScript setup and dependencies

-- Only intended for personal use
